# EEK StackChan Overview

This file provides guidance to users when working with code in this repository.

## Introduction

The StackChan firmware can be replaced with Arduino-based software where its Servo motion and Camera features can be controlled wirelessly by either the M5Stick StackChan controller that is optionally included with the StackChan hardware or by the EEK. Control of the StackChan when running the Tello drone simulator firmware is also possible from the M5Stick controller.
